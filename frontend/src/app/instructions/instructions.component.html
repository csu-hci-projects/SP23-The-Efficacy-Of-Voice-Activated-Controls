Welcome!
Purpose:

This experiment is to measure the efficacy and accuracy of voice control versus mouse input.

Contents:

There will be a two tests followed by a survey. Each test will be administered by playing the classic game Simon Says.

Game Rules:

There are four buttons arranged in a square. Each button has a different color and tone associated with it. When you are ready, click Start Game to begin. The game will display a sequence of colors in a specific order. It is your job to memorize and select those colors in whatever order it asks as accurately and quickly as possible. The game will start with one color and increase by 1 with each successful iteration. You have 3 attempts to achieve a high score round of a given sequence. With each incorrect entry, you lose an attempt. After 3 attempts you will move on to the next portion of the experiment.
